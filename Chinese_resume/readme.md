
## 中文简历筛选数据集（部分 IT 行业）

> Source: [点击查看](https://tianchi.aliyun.com/dataset/201566)

模拟生成的结构化简历数据，共 **5000 条记录、34 个字段**，每份简历均已标注筛选结果（`通过` / `不通过`），可用于构建简历筛选模型或分析影响录用的关键因素。数据均为模拟生成，不涉及真实个人信息。

### 字段说明

| 类别         | 字段                                   | 说明                                                                                |
| ------------ | -------------------------------------- | ----------------------------------------------------------------------------------- |
| **基本信息** | 简历编号、姓名、性别、年龄、电话、邮箱 | 简历编号为唯一标识                                                                  |
| **求职意向** | 意向岗位                               | 算法工程师、测试工程师等共 10 类岗位                                                |
| **教育背景** | 学历层次、院校类别、专业类别、英语水平 | 如：本科、普通高校、计算机类、英语四级                                              |
| **技能栈**   | 编程语言及熟练度                       | Java、Python 等，附对应熟练度                                                       |
|              | 各技术领域及熟练度                     | 前端技术、后端技术、数据库、云计算/运维、数据与算法、移动开发、测试工具，各附熟练度 |
| **工作经验** | 小型 / 中型 / 大型企业工作经验         | 以时间段表示（如 1―3 年）                                                           |
| **项目经验** | 小规模 / 中规模 / 大规模项目           | 以项目数量表示                                                                      |
| **筛选结果** | 通过 / 不通过                          | 目标变量（标签）                                                                    |
